# Uranium 235 — Field Manual (estado del proyecto)

## Qué es
Sitio de una sola página (`index.html`) con el manual de campo del clan de HLL, deployado en GitHub Pages.

## Última modificación
Se agregó una capa nueva de "macro" pre-partido, entre Roles y Comunicación:

1. **Pre-match** — qué se define antes de jugar (roster, stratsketch, aperturas).
2. **Leer el roster** — columnas (Inicio, Comando, INF1/2/3, FLEX AT, FLEX, RECON, defensa PR1/DEF/PR2/WIDE).
3. **Leer el stratsketch** — banderas por color de grupo, garrisons, L/M/R de tanque, círculos rojos de recon, líneas punteadas.
4. **Aperturas y 3-2/2-3** — plan de arranque y esquemas de bloque de color según se gane o pierda el punto medio.

El resto de las secciones (Comunicación, Proactividad, Datos, Infantería, Tanquista, Reglas) quedaron igual, solo se renumeraron.

## Archivos en este paquete
- `index.html` — versión completa y actualizada del sitio.
- `media/premarch/` — 12 imágenes usadas en las 4 secciones nuevas.
- `Prematch_Uranium235.docx` — las 4 secciones nuevas en Word, para leer sin abrir el sitio.
